# GenAI BCG X 🚀

This repository contains the deliverables, code, and analysis completed during the **BCG X Generative AI Virtual Job Simulation** hosted on Forage. The project focuses on leveraging financial data extraction and Large Language Models (LLMs) to build an intelligent corporate analysis tool.

---

## 📌 Project Overview

The simulation replicates the real-world workflow of a BCG X Data Scientist / AI Engineer, transitioning from raw data processing to an AI-driven user experience.

* **Task 1: Financial Data Extraction & Analysis** Automated the retrieval and parsing of critical financial metrics (Revenue, Net Income, Total Assets, etc.) from corporate SEC filings and statements using `pandas`.
  
* **Task 2: AI-Driven Financial Chatbot Prototype** Developed an intelligent Retrieval-Augmented Generation (RAG) style chatbot application designed to accurately answer financial queries, eliminate hallucinations, and maintain conversational context.
